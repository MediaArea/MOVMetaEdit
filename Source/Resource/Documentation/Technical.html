Technical metadata editor

After having drag and dropped (or from the "File" menu) some files, the technical metadata view displays information about each file in a table.
Then it is possible to edit, delete, and save the metadata.

In case a file is not supported, the Filename column is in red, and the metadata is not modifiable.

Supported fields

  • Clean aperture dimensions (clef), "WxH" format.
  • Production aperture dimensions (prof), "WxH" format.
  • Aperture pixels size (enof), "WxH" format.
  • Pixel aspect ratio (pasp), "x:y" format.
  • Width scale, real number.
  • Field handling parameter (fiel), "fields,detail" format.
  • Color parameter (colr), "primaries,transfer,matrix" format.
  • Gamma (gama), real number.
  • Clean aperture (clap), "with:d,height:d,hOffset:d,vOffset:d" format.
    (':d' (denominator) are optionnal')
  • Channel description (chan), editor.
A cell in bold means that the field is valid and the file can be saved (click on the "Save" button or the "Save" item in the "File" menu)
