MOV MetaEdit Help

About this Software

MOV MetaEdit was developed by MediaArea, and sponsored by Ad-ID.

This tool permits the embedding and editing of metadata in MOV (Apple QuickTime) or MP4 (ISO/IEC 14496-14 a.k.a. MPEG-4 Part 14) files.
It is currently focused on Universal Ad ID metadata but could be expanded on request.
